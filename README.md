# Modernizr
* [Modernizr](https://modernizr.com/) with npm.

```bash
# Install
npm install -g modernizr

# create 'modernizr.json' config file
modernizr -c modernizr.json 
```

This will create  a modernizr.js at same directory with the custom modernizr.
